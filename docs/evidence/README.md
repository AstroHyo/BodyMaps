# Evidence

This directory stores BodyMaps proof-of-work assets that are safe to show in the repository README.

## Current UI Flow

The current checked-in evidence lives in `docs/evidence/app-flow/`.

- `01-home.png`: upload screen.
- `03-runpod-visualization-default.png`: four-pane visualization loaded from a real RunPod inference result.
- `05-sidebar-controls-adjusted.png`: organ visibility and window/level/opacity controls adjusted.
- `06-segmentation-data-modal.png`: segmentation data modal.
- `07-axial-fullscreen.png`: axial viewport fullscreen.
- `08-sagittal-fullscreen.png`: sagittal viewport fullscreen.
- `09-coronal-fullscreen.png`: coronal viewport fullscreen.
- `10-3d-rotated-fullscreen.png`: rotated 3D viewport.
- `bodymaps-runpod-visualization.gif`: inline README demo preview generated from the app-only recording.
- `bodymaps-runpod-visualization.mp4`: short app interaction recording.

The older root-level evidence screenshots and recordings were removed because they included non-app desktop context. New evidence should only capture the BodyMaps application surface.
